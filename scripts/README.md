# Scripts de inserción masiva

Los scripts de seed se generan en el Día 2 del proyecto.
Ejecutar en este orden después de tener MongoDB corriendo:

1. npm run seed:categorias   → 50 categorías
2. npm run seed:productos    → 300.000 productos
3. npm run seed:usuarios     → 500.000 usuarios
4. npm run seed:pedidos      → 699.950 pedidos

Total esperado: 1.500.000 documentos
